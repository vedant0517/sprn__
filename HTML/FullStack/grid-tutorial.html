  • ‚úÖ Use consistent z-index scale (1, 10, 100, 1000)
  • ‚úÖ Only positioned elements (relative, absolute, fixed) use z-index
  • ‚úÖ Grid items automatically create stacking context
  • ‚úÖ Higher z-index values appear in front
  • ‚ùå Avoid random large numbers (z-index: 99999)
